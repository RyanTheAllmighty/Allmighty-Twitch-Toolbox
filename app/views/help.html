Getting Started

To get started you'll need a few plugins installed in OBS (OBS MultiPlatform not supported). Please download and install them into your OBS:

  • CLR Browser Source
  • OBS Remote

You'll also need foobar2000 and a coupple plugins for it in order to access the foobar related things:

  • foobar2000
  • foo_np_simple
  • foo_httpcontrol
  • foo_httpcontrol ajquery

The settings for foo_np_simple can be accessed within foobar under File -> Preferences -> Tools -> Now Playing Simple and should be set as encoding UTF-8 and should only respond to the 'On New Track' event. The Formatting string section should only contain '%path%' without the quotes.

Once installed and OBS and foobar2000 have been restarted you can then go to the 'Settings' tab in this application and setup your settings.

Once everything is setup, a restart of the application is needed so it can start to pull in all the information it needs to get started.

Scenes

To get started with setting up the necessary scenes in OBS, take a look at the below list of the url's of the scenes and their width and height.

These size settings are based off a 1280x720 broadcasting resolution. If you're using anything but that, you will need to resize your notifications within OBS to suit.

If you want to have these playing across multiple scenes and not reloading everytime you switch scene, then you should set them up as global sources.

  • Now Playing Scene (175x300): http://localhost:28800/scenes/nowplaying/
  • Alerts Scene (600x50): http://localhost:28800/scenes/alerts/
  • Timers Scene (500x125): Click 'Copy Link' on the timer on the Timers tab for the link

Overlays

Overlays are groups of different elements which are shown in a 1280x720 resolution and 'overlayed' over your game capture in OBS.

Overlays are based off a 1280x720 broadcasting resolution and are best to be added as global scenes so that their state, transitions etc, carry over between scenes.

Below is the list of overlays and their url's to be used in OBS.

  • Default Overlay (Alerts and Now Playing Notifications): http://localhost:28800/overlays/default/

Tools

This application comes with some tools which aim to help make life easier. Tools can be accessed by right clicking on the applications icon in the taskbar and clicking the tool.

The microphone status tool brings up a resizable window (which remembers it's last position and size) and shows if the current scene in OBS has it's microphone muted or not.

The tool also provides a hotkey (which can be set in the settings) which when pressed will toggle the current scenes microphone muting. This is best used with the 'Per Scene Volume' plugin which comes with OBS. If you don't want to be able to toggle a scenes microphone mute status you can add '[NOMUTE]' to the name of the scene or as the name of a source in the scene (see image).
